Tidy-Viewer is organized as a Cargo workspace with three main components:

  • tidy-viewer-core - Shared core library with data type inference and formatting logic
  • tidy-viewer-cli - Command-line interface for direct file processing
  • tidy-viewer-py - Python bindings using PyO3 for integration with Python workflows

‚ú® Key Features

  • Multi-format Support for CSV, TSV, PSV, Parquet, and Arrow files
  • Smart Formatting Intelligent significant figure handling and column width calculation
  • Unicode Support Full Unicode character width calculation and truncation
  • Type Inference Automatic detection of data types (numbers, dates, times, logical)
  • Cross-platform Works on Windows, macOS, and Linux
  • Python Integration Native Python bindings for data science workflows
